# 03 Functional Requirements

## Admin Panel
- **Dashboard**: View high-level statistics (patients, doctors, appointments).
- **Patient Management**: Add, update, view, and delete patient records.
- **Doctor Management**: Manage accounts, update details, assign specializations, manage availability.
- **Staff Management**: Manage employee records (nurses, receptionists), assign roles, manage schedules.
- **Medication Management**: Manage pharmacy inventory, track stock, check expiration dates.
- **Billing Management**: View bills, generate invoices, confirm payments, track transactions.
- **User Management**: Create accounts, assign roles/permissions, access control.

## Patient Services
- **Registration**: Create unique patient profiles with personal/medical info.
- **Profile Management**: Update personal contact and medical data.
- **Appointment Booking**: View doctors, check availability, book visits.
- **Medical Records**: View historical diagnoses, treatments, and prescriptions.
- **Billing**: View personal hospital bills and payment history.
- **Notifications**: Receive reminders for appointments and updates.

## Doctor Services
- **Doctor Dashboard**: Overview of daily activities and upcoming appointments.
- **Patient Record Access**: View medical histories and previous treatments.
- **Appointment Management**: Manage schedule, update appointment status (completed/canceled).
- **Diagnosis & Treatment**: Record consultation notes, diagnoses, and medical advice.
- **Prescription Management**: Create and manage patient prescriptions.
- **Medical Reports**: View and add notes to laboratory test results.

## General
- **Authentication**: Secure login/logout based on roles.
- **Search**: Multi-criteria search for patients, doctors, and records.
- **Audit Logging**: Track system activities and data changes for security.
